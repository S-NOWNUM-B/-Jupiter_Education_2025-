## О проекте

> **Практический подход к анализу данных через решение реальных задач**

Этот репозиторий содержит задания и проекты по курсу анализа данных. От базовой работы с массивами до создания сложных визуализаций и статистического анализа данных с использованием современных библиотек Python.

***

## Быстрый старт

### Клонирование репозитория

    git clone <repository-url>
    cd Jupiter

### Установка зависимостей

    pip install -r requirements.txt

### Запуск Jupyter Notebook

    jupyter notebook
